在网页中引入Github-Star

Posted by Superman on 2017-02-10

最近喜欢上折腾点小东西，顺便练练技术，熟悉下开源社区的玩法。经常会看到网页中引入github repository 的start的图标和“fork me on github”的丝带，长👇这样：


有三种方式可以实现在页面、markdown引入star。

方式一：iframe嵌入

将user=superman66换成你的github username，repo=vue2.x-douban换成你想要显示的repository即可。
iframe方式适用于在html页面中引入。

方式二：使用img.shields.io提供的服务

将superman66/vue2.x-douban换成your-github-username/your-repo即可。
此方式适用于在markdown中引入。

方式三：Fork me on Github

Github 提供了多种颜色的Ribbons。

同样的，写上你自己的githu username即可。
此方式适用于在html页面中引入。
具体更多的样式，前往官网查看。
